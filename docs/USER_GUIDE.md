# User Guide

Open the analysis tab, click a start point on the map, choose:

- Travel mode: walk, bike, or drive.
- Cost type: time or distance.
- Limit: minutes for time or meters for distance.

The result shows reachable roads, a service area polygon, area size, road length, reached nodes, and intersecting districts.

Layer controls can toggle the service area polygon and reachable roads. Export downloads the service area GeoJSON with the selected parameters in the filename.
